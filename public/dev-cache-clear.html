🧹 Development Cache Clear

Use this page to manually clear browser caches during development.

Cache Information:

Instructions:

  • Always use Hard Refresh during development: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
  • Disable cache in DevTools: Open DevTools → Network tab → Check "Disable cache"
  • For fresh builds: Run npm run build:fresh or php artisan frontend:clear-cache
  • Check browser console for any cache-related messages
